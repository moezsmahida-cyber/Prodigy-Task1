# Task 01 - Data Visualization

## Objective

To create visualizations (Histogram and Bar Chart) using a dataset.

## Tools Used

* Python
* Pandas
* Matplotlib

## Dataset

The dataset contains both numerical and categorical data.

## Visualizations

### Histogram

Displays distribution of numerical data.

### Bar Chart

Displays distribution of categorical data.

## Output

* histogram.png
* bar_chart.png

## How to Run

1. Install libraries:
   pip install pandas matplotlib

2. Run the code:
   python task1.py
